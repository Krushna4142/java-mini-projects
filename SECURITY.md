# 🔐 Security Policy

## 📌 Supported Versions

This project is for learning purposes, but security improvements are always welcome.

---

## 🚨 Reporting a Vulnerability

If you discover a security issue:

1. Do NOT open a public issue
2. Contact directly via GitHub

GitHub: https://github.com/Krushna4142

---

## 🔍 What to Include

- Description of the issue
- Steps to reproduce
- Possible impact

---

## ⚡ Response

All valid reports will be reviewed and addressed.

---

## 💡 Note

This is a learning project, but we aim to follow **best practices** wherever possible.
