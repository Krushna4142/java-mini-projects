# 🎯 Mock Interview Q&A — Java Fundamentals

These questions help revise concepts learned during the first **21 days of Java practice**.

---

# 1️⃣ What is Java?

Java is a **high-level, object-oriented programming language** used to build platform-independent applications.

Java programs run on the **Java Virtual Machine (JVM)**.

---

# 2️⃣ What is Object-Oriented Programming?

OOP is a programming paradigm based on **objects and classes**.

Main principles:

• Encapsulation  
• Inheritance  
• Polymorphism  
• Abstraction

---

# 3️⃣ What is Encapsulation?

Encapsulation means **wrapping data and methods inside a class** and controlling access to that data.

Example:

Private variables with public getters/setters.

---

# 4️⃣ What is the difference between == and .equals()?

`==` compares memory references.

`.equals()` compares actual values.

Example:

String a = "Java";
String b = "Java";

a.equals(b) → true

---

# 5️⃣ What is an Array?

An array stores **multiple values of the same type** in a single variable.

Example:

int numbers[] = {10,20,30};

---

# 6️⃣ What is a Loop?

A loop allows a block of code to run multiple times.

Types:

• for loop  
• while loop  
• do-while loop

---

# 7️⃣ What is File Handling?

File handling allows programs to **read and write data to files**.

Important classes:

• File  
• FileWriter  
• FileReader  
• BufferedReader

---

# 8️⃣ What is a Menu Driven Program?

A program where users choose actions from a **menu of options**.

Example:

1. Add Record
2. Delete Record
3. View Records

---

# 9️⃣ What is a 2D Array?

A 2D array represents a **table or grid** with rows and columns.

Example:

int matrix[][] = new int[3][3];

Used in:

• Game boards  
• Matrix operations  
• Grid systems

---

# 🔟 What is the Scanner Class?

Scanner is used to **take user input** in Java programs.

Example:

Scanner sc = new Scanner(System.in);
int num = sc.nextInt();

---

# 🚀 Final Tip

During interviews:

Do not just memorize answers.

Always explain:

• concept  
• example  
• real-world use

That shows deeper understanding.
