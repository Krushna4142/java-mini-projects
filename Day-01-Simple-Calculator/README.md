## â“ Problem Statement

Create a **console-based calculator** that:

- Takes two numbers as input
- Allows the user to choose an operation
- Performs basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division

The program should handle invalid choices gracefully.
