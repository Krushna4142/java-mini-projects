# 🎯 Mock Interview Q&A

These questions help revise the concepts learned in the first 21 days.

### Q1. What is the difference between == and .equals()?

`==` compares memory references while `.equals()` compares actual values.

---

### Q2. What is encapsulation?

Encapsulation is wrapping data and methods inside a class and restricting direct access.

---

### Q3. Why are arrays useful?

Arrays allow storing multiple values of the same type in a single structure.

---

### Q4. What is a menu-driven program?

A program that allows users to choose actions through a list of options.

---

### Q5. Why is file handling important?

File handling allows programs to store and retrieve data permanently.
